Діаграма прецедентів
@startuml

actor "Administrator" as Administrator actor "Editor" as Editor actor "User" as User

usecase "ADM_1\nКерувати текстовими файлами" as ADM_1 usecase "ADM_2\nКерувати редакторами\n та редагуванням" as ADM_2

Administrator -r-> ADM_2 Administrator -r-> ADM_1

usecase "EDITOR_1\nСтворювати та редагувати\n розмітку даних" as EDITOR_1

Editor -r-> EDITOR_1

usecase "USER_1\nРеєструватися та авторизуватися\n в системі" as USER_1 usecase "USER_2\nШукати, переглядати\n та завантажувати файли з сервера" as USER_2 usecase "USER_3\nПодавати запит на\n права редактора" as USER_3

User -r-> USER_3 User -r-> USER_2 User -r-> USER_1

Administrator -d-|> Editor Editor -d-|> User

@enduml

@startuml

actor "User" as User

usecase "USER_1\nРеєструватися та\n авторизуватися у системі" as USER_1 #91FF5B usecase "USER_2\nШукати, переглядати\n та завантажувати файли з сервера" as USER_2 #91FF5B usecase "USER_3\nПодавати запит на\n право редагування" as USER_3 #91FF5B

@enduml

usecase "ADM.VIEW_HISTORY\nПереглянути історію змін файлу" as ADM.VIEW_HISTORY usecase "ADM.VIEW_BRANCH\nПереглянути гілку змін файлу" as ADM.VIEW_BRANCH

ADM.FILE_UPLOAD ..> ADMIN_1 :extends ADM.ADD_EDITOR ..> ADMIN_1 :extends ADM.DELETE_EDITOR ..> ADMIN_1 :extends ADM.FILE_DELETE ..> ADMIN_1 :extends

ADM.ADMT_TXT_ANNOTATION_FINAL ..> ADMIN_2 :extends ADM.ADMT_TXT_ANNOTATION_CHNG ..> ADMIN_2 :extends

Сценарії використання для користувача
ID: USER.REG

НАЗВА: Реєстрація в системі

УЧАСНИКИ: Користувач, Система

ПЕРЕДУМОВИ: Користувач не зареєстрований у системі

РЕЗУЛЬТАТ: Новий обліковий запис

ВИКЛЮЧНІ СИТУАЦІЇ: Відхилення запиту на реєстрацію (USER.REG_DENY)

ОСНОВНИЙ СЦЕНАРІЙ:

@startuml

|Користувач| start :Вводить реєстраційні дані;

|Система| :Перевіряє передані реєстраційні данія; note right #ffaaaa Можлива USER.REG_DENY end note

:Cтворює новий обліковий запис, використовуючи введені дані;

:Надає користувачу інформацію про створення облікового запису;
|Користувач|
:Завершує взаємодію з системою;

stop;
@enduml

:Завершує взаємодію з системою;

@enduml
